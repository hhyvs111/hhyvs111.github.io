1. Two Sum

Posted on 2018-11-26 Edited on 2020-04-14 In 算法

要开始刷算法题了，昨天去头条笔试被吊打了，现在的自己原来这么菜。

那么就从LeetCode开始刷吧。

题目描述

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.
Example:

Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

思路

就是在数组里找两个数是否是相加等于这个目标值。

直接想到的方法就是遍历，然后判断是否相等。

代码

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        vector<int> ans;
        int size = nums.size();
        for(int i = 0; i < size; i++){
            for(int j = i + 1 ;j < size; j++){
                if(nums[i] + nums[j] == target){
                    ans.push_back(i);
                    ans.push_back(j);
                    return ans;
                }
            }
        }
    }
};

另一种方法

我看排行第一的用的是HashMap。

完全不知道HashMap原理，直接贴出来看看。

class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
      std::unordered_map<int, std::size_t> tmp;  
      for (std::size_t i = 0; i < nums.size(); ++i) {
        if (tmp.count(target - nums[i])) {
            return {tmp[target - nums[i]], i};
        }
        tmp[nums[i]] = i;
    }
        return {0,0};

    }
};
