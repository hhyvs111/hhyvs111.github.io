指向指针的引用

发表于 2018-04-13 更新于 2020-04-14 分类于 C++

在前面的笔记我们有写道，引用的作用&。

就是给你引用的变量起了另外一个名字，对这个进行操作也会改变你所引用的变量。

引用本身不是一个对象，因此不能定义指向引用的指针。但指针是对象，所以存在对指针的引用。

int i = 42; int *p; int *&r = p; //r是一个对指针p的引用 r = &i; //这里要注意为什么是空r = &i，因为r也是一个指针，没有前置操作符的指针r = 某一个值，是指向操作，所以指向的东西要加&。这里就是令P指向i。 *r = 0; //解引用r得到i，也就是p指向的对象，将i的值改为0；

这里要理解r的类型到底是什么，最简单的方法是从右向左阅读r的定义（←阅读）。离变量名最近的符号（此例中是&r的符号r）对变量的类型有最直接的影响，因此r是一个引用。

声明符的其余部分用以确定r引用的类型是什么，此例中的符号*说明r引用的是一个指针。最后，声明的基本数据类型部分指出r引用的是一个int指针。
