3 创建System V IPC键

3.1 ftok函数

#include <sys/types.h>
#include <sys/ipc.h>
key_t ftok(const char *pathname, int proj_id);

三种类型的System V IPC使用key_t作为它们的名字。这个时候就可以用该值创建相应的进程通信文件，比如：

int msgget(key_t key, int msgflg);

flg位和Posix类似。

3.2 指定key

除了调用ftok函数，还可以指定key为IPC_PRIVATE，这将保证会创建一个新的、唯一的IPC对象。
