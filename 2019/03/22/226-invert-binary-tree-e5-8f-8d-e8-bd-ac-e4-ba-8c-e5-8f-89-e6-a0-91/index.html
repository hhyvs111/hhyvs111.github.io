226. Invert Binary Tree 反转二叉树

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

反转二叉树。

Input:

     4
   /   \
  2     7
 / \   / \
1   3 6   9
Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1

题解

比较经典的一道题。

递归版

先定义左右节点等于递归后的左右节点，然后交换他们的值。

class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if(!root) return NULL;
        TreeNode* right = invertTree(root->right);
        TreeNode* left = invertTree(root->left);
        root->left = right;
        root->right = left;

        return root;
    }
};

迭代版

利用栈，先把左右子树存进去，然后交换。

TreeNode* invertTree(TreeNode* root) {
    std::stack<TreeNode*> stk;
    stk.push(root);

    while (!stk.empty()) {
        TreeNode* p = stk.top();
        stk.pop();
        if (p) {
            stk.push(p->left);
            stk.push(p->right);
            std::swap(p->left, p->right);
        }
    }
    return root;
}
