283. Move Zeroes 移动0

Posted on 2019-03-22 Edited on 2020-04-14 In 算法

题意

给一串数组，将里面所有的0都移动到后面，而且不能改变原来非0数字的前后顺序。原地交换，不能用额外的空间。

Input: [0,1,0,3,12]
Output: [1,3,12,0,0]

题解

用一个计数器，每次遇到非0就把计数器对应数组的位置的值置为非0值，然后再用0填充后面的数字。

class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        //找到0则互换，必须原地互换
        int n = nums.size() - 1;
        int cnt = 0;
        for(int i = 0;i <= n;i++)
        {
            if(nums[i] != 0)
            {
                nums[cnt++] = nums[i];
            }
        }
        for(int i = cnt; i <= n;i++)
            nums[i] = 0;
    }
};
