定义

变量定义

这里的函数变量如果是同一类型可以连续定义，比如下面这种

func add(x, y int) int {
return x + y
}

个人感觉比c++方便不少哦，不用一个变量写一个int。

多值返回

函数可以返回多个值，妈耶这个比c++好多了。

func swap(x, y string) (string, string) {
    return y, x
}
func main() {
    a, b := swap("hello", "world")
    fmt.Println(a, b)
}

这里的赋值运算不一样 是这个:=，而且也不用提前定义变量，可以直接用a，b。应该是可以直接解析成string类型吧。

要特别注意这个赋值运算符。

占位符

如果开发者只对该函数其中的某几个返回值感兴趣的话，也可以直接用下划线作为占位符来 忽略其他不关心的返回值。下面的调用表示调用者只希望接收lastName的值，这样可以避免声 明完全没用的变量：

_, _, lastName, _ := getName()

命名返回值

返回值可以被命名，像是变量那样使用。

func split(sum int) (x, y int) {
    x = sum * 4 / 9
    y = sum - x
    return
}
func main() {
    fmt.Println(split(17))
}

返回值可以直接在函数体里操作，有点奇怪。然后直接return，函数参数后面的括号就是返回值。

定义变量

这里的变量就是var， 中间是变量名 后面是类型。

var c, python, java bool
func main() {
    var i int
    fmt.Println(i, c, python, java)
}

默认int是0，然后bool是false。所以输出的是0和三个false。

初始化变量

变量定义可以包含初始值，每个变量对应一个。 如果初始化是使用表达式，则可以省略类型；变量从初始值中获得类型。

package main
import "fmt"
var i, j int = 1, 2
func main() {
    var c, python, java = true, false, "no!"
    fmt.Println(i, j, c, python, java)
}

短变量声明

上面的 := 是短变量声明，可以省略类型。但是不能再函数外使用，也就是不能当全局变量。

类型转换

Go只能显式转换，和C不同不能隐式转换。

func main() {
    var x, y int = 3, 4
    var f float64 = math.Sqrt(float64(x*x + y*y))
    var z int = int(f)
    fmt.Println(x, y, z)
}

常量

月一样用const，但是不能用:=来定义，只能用=来定义。

const Pi = 3.14
func main() {
    const World = "世界"
    fmt.Println("Hello", World)
    fmt.Println("Happy", Pi, "Day")
    const Truth = true
    fmt.Println("Go rules?", Truth)
}

控制结构

for

Go 只有一种循环结构——for 循环，而且不能用括号，其他的和c和java一样，必须使用中括号{}

func main() {
    sum := 0
    for i := 0; i < 10; i++ {
        sum += i
    }
        fmt.Println(sum)
}

也可以让前置为空，有点像是for( ; ;sum < 100)

func main() {
    sum := 1
    for sum < 1000 {
        sum += sum
    }
    fmt.Println(sum)
}

For就是while一样，如果什么都没有就是死循环

func main() {
    for {
    }
}

if

也是和for一样，不能加括号。

func sqrt(x float64) string {
    if x < 0 {
        return sqrt(-x) + "i"
    }
    return fmt.Sprint(math.Sqrt(x))
}

但是比C里面方便的是在判断之前可以实行一个简单的语句，且作用域仅限于if范围内。

func pow(x, n, lim float64) float64 {
    if v := math.Pow(x, n); v < lim {
        return v
    }
        return lim
    }
func main() {
    fmt.Println(
    pow(3, 2, 10),
    pow(3, 3, 20),
    )
}

这里定义了一个v，然后进行一顿操作再判断是否小于lim。

else

一样是if-else结构，但是好像没有else if？

func pow(x, n, lim float64) float64 {
    if v := math.Pow(x, n); v < lim {
        return v
    } else {
        fmt.Printf("%g >= %g\n", v, lim)
    }
    // 这里开始就不能使用 v 了
    return lim
}

switch

和c里差不多，但是和go里的if一样，也可以定义一个前置声明。

func main() {
    fmt.Print("Go runs on ")
    switch os := runtime.GOOS; os {
        case "darwin":
            fmt.Println("OS X.")
        case "linux":
            fmt.Println("Linux.")
        default:
            // freebsd, openbsd,
            // plan9, windows...
            fmt.Printf("%s.", os)
    }
}

但是这里不用加break，不会像c里那样如果不加break还会顺序执行。

没有条件的switch

这一构造使得可以用更清晰的形式来编写长的 if-then-else 链。

func main() {
    t := time.Now()
    switch {
        case t.Hour() < 12:
            fmt.Println("Good morning!")
        case t.Hour() < 17:
            fmt.Println("Good afternoon.")
        default:
            fmt.Println("Good evening.")
    }
}

就是类似于这个if else if之类的，不用switch来判断条件

defer

会延迟函数的执行知道上层函数返回。

func main() {
    defer fmt.Println("world")
    fmt.Println("hello")
}

延迟调用的参数会立刻生成，但是在上层函数返回前函数都不会被调用。 但是这个有什么用呢？谷歌了一下发现用处挺大的。

  • 比如用来释放资源：文件资源的释放会在函数调用结束之后借助 defer 自动执行，不需要时刻记住哪里的资源需要释放，打开和释放必须相对应。

  • 用来当栈使用，会按照后进先出的顺序调用被延迟的函数调用

    func main() {

    fmt.Println("counting")
for i := 0; i < 10; i++ {
    defer fmt.Println(i)
}
fmt.Println("done")

    }

101. Symmetric Tree 镜像二叉树

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

判断一颗二叉树是否是镜像二叉树，也就是中心对称。

For example, this binary tree [1,2,2,3,4,4,3] is symmetric:

    1
   / \
  2   2
 / \ / \
3  4 4  3
But the following [1,2,2,null,3,null,3] is not:
    1
   / \
  2   2
   \   \
   3    3

题解

递归版

利用递归思想, root相等且递归t1的左节点等于t2的右节点相等，那么才返回true，如果一个为空，另一个不为空直接返回false。

class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        return mirror(root,root);
    }
    bool mirror(TreeNode* t1, TreeNode* t2){
        if(t1 == NULL && t2 == NULL) return true;
        if(t1 == NULL || t2 == NULL) return false;
        return (t1->val == t2->val) && mirror(t1->left,t2->right) && mirror(t1->right, t2->left);
    }
};

迭代版

用栈做，push两次根节点，然后取出来判断。再依次push进去t1的左节点和t2的右节点，因为相近的就是要比较的，每次弹出两个来比较。

class Solution {
public:
    vector<int> nums;
    vector<int> numsRev;
    bool isSymmetric(TreeNode* root) {
        if(root == NULL)
            return true;
        //左右一个栈，同时进行递归
        //镜面原理，将一个树分为两颗子树，他们符合一个规律。左子树等于另一个树的右子树。
        stack<TreeNode*> s;
        s.push(root);
        s.push(root);
        while(!s.empty())
        {
            TreeNode* t1 = s.top();
            s.pop();
            TreeNode* t2 = s.top();
            s.pop();
            //如果都为空则继续，可能有空节点
            if(t1 == NULL && t2 == NULL) continue;
            if(t1 == NULL || t2 == NULL) return false;
            if(t1->val != t2->val) return false;
            s.push(t1->left);
            s.push(t2->right);
            s.push(t1->right);
            s.push(t2->left);
        }
        return true;
    }
};

53. Maximum Subarray 最大的连续子数组和

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

给一串数字，求出连续的子数组的最大和。用O(n)解决。

Input: [-2,1,-3,4,-1,2,1,-5,4],
Output: 6
Explanation: [4,-1,2,1] has the largest sum = 6.

题解

依次累加，定义一个当前sum和最大sum。

  • 如果nowSum加了这个值是大于nums[i]的，那么就加，因为是正向的；
  • 如果nowSum加了这个值比当前的值还要小，那么这个序列是成负向的，直接把标号移动到nums[i]的位置。

因为要尽量跳过负数，所以如果减到0那么就要重新累加。

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int nowSum = nums[0];
        int maxSum = nowSum;
        for(int i = 1; i < nums.size();i++)
        {
            nowSum = (nowSum + nums[i])  >= nums[i]? (nowSum + nums[i]) : nums[i];
            maxSum = max(maxSum,nowSum);
        }
        return maxSum;
    }
};

70. Climbing Stairs 爬楼梯

发表于 2019-03-22 更新于 2020-04-14 分类于 C++

题意

给你一个高度为n的台阶，每次你可以爬1层台阶，或者爬2层台阶。有多少种爬楼梯的方法。

Example 1:
Input: 2
Output: 2
Explanation: There are two ways to climb to the top.
1. 1 step + 1 step
2. 2 steps

Example 2:
Input: 3
Output: 3
Explanation: There are three ways to climb to the top.
1. 1 step + 1 step + 1 step
2. 1 step + 2 steps
3. 2 steps + 1 step

题解

动态规划，初始化1为1种，2为两种。dp[i]为前面两个dp的和。

class Solution {
public:
    int climbStairs(int n) {
        //据说是最简单的dp？
        if(n == 1) return 1;
        vector<int> dp (n+1);
        dp[1] = 1;
        dp[2] = 2;
        //从前面推后面
        for(int i = 3 ; i <= n;i++)
        {
            dp[i] = dp[i-1] + dp[i-2];

        }
        return dp[n];
    }
};

121. Best Time to Buy and Sell Stock 股票最佳买卖 — 单次买卖

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

一个数组，代表股票当前价格，买入和买出。怎么操作能使收益最大？只有卖掉当前股票才能买。

Example 1:
Input: [7,1,5,3,6,4]
Output: 5
Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5.
             Not 7-1 = 6, as selling price needs to be larger than buying price.

Example 2:
Input: [7,6,4,3,1]
Output: 0
Explanation: In this case, no transaction is done, i.e. max profit = 0.

题解

菜鸡版

因为只能买卖一次，求出最大的值，然后求出最小的值。当最小值出现的时候，如果最大值的下标小于最小值的下标，那么重置最大值

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        if(prices.size() <= 1) return 0;
        int maxProfit = 0;
        int minPrice = prices[0], maxPrice = 0;
        int indexMax = 0, indexMin = 0;
        for(int i = 0;i < prices.size();i++)
        {
            if(prices[i] > maxPrice)
            {
                maxPrice = prices[i];
                indexMax = i;
            }
            if(prices[i] < minPrice)
            {
                minPrice = prices[i];
                indexMin = i;
                //当最小值出现的时候，如果最大值的下标小于最小值的下标，那么重置最大值
                if(indexMin > indexMax)
                    maxPrice = 0;
            }
            if(maxPrice - minPrice > maxProfit)
                maxProfit = maxPrice - minPrice;
        }
        return maxProfit;
    }
};

大佬版

遍历一次数组，用一个变量记录遍历过数中的最小值，然后每次计算当前值和和这个最小值之间的差值作为利润，然后每次选较大的利润来更新，当遍历完成后当前利润即为所求。

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int res = 0, buy = INT_MAX;
        for (int price : prices) {
            buy = min(buy, price);
            res = max(res, price - buy);
        }
        return res;
    }
};

448. Find All Numbers Disappeared in an Array 找出数组里消失的值

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

给你一个数组，其值是从1到n（数组长度）的数，但是有一些数没有出现，让你找出这些数。

Input:
[4,3,2,7,8,2,3,1]

Output:
[5,6]

题解

比如这里5，6没有出现，但是2和3重复出现了两次。可以利用这个性质求解，将数字放到数组中正确位置。然后再循环一遍数组，当数组的位置与值不对应，就是要找的值。

空间复杂度O(1)

class Solution {
public:
    vector<int> findDisappearedNumbers(vector<int>& nums) {
        vector<int> ans;
        for(int i = 0 ; i < nums.size();i++)
        {
            //将数组放回原来的位置
            if(nums[i] != nums[nums[i] - 1])
            {
                swap(nums[i], nums[nums[i] - 1]);
                --i;
            }
        }
        for(int i = 0 ; i < nums.size();i++)
        {
            if(nums[i] != i + 1)
                ans.push_back(i+1);
        }
        return ans;
    }
};

283. Move Zeroes 移动0

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

给一串数组，将里面所有的0都移动到后面，而且不能改变原来非0数字的前后顺序。原地交换，不能用额外的空间。

Input: [0,1,0,3,12]
Output: [1,3,12,0,0]

题解

用一个计数器，每次遇到非0就把计数器对应数组的位置的值置为非0值，然后再用0填充后面的数字。

class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        //找到0则互换，必须原地互换
        int n = nums.size() - 1;
        int cnt = 0;
        for(int i = 0;i <= n;i++)
        {
            if(nums[i] != 0)
            {
                nums[cnt++] = nums[i];
            }
        }
        for(int i = cnt; i <= n;i++)
            nums[i] = 0;
    }
};

226. Invert Binary Tree 反转二叉树

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

反转二叉树。

Input:

     4
   /   \
  2     7
 / \   / \
1   3 6   9
Output:

     4
   /   \
  7     2
 / \   / \
9   6 3   1

题解

比较经典的一道题。

递归版

先定义左右节点等于递归后的左右节点，然后交换他们的值。

class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if(!root) return NULL;
        TreeNode* right = invertTree(root->right);
        TreeNode* left = invertTree(root->left);
        root->left = right;
        root->right = left;

        return root;
    }
};

迭代版

利用栈，先把左右子树存进去，然后交换。

TreeNode* invertTree(TreeNode* root) {
    std::stack<TreeNode*> stk;
    stk.push(root);

    while (!stk.empty()) {
        TreeNode* p = stk.top();
        stk.pop();
        if (p) {
            stk.push(p->left);
            stk.push(p->right);
            std::swap(p->left, p->right);
        }
    }
    return root;
}

104. Maximum Depth of Binary Tree 二叉树的最大深度

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

求二叉树的最大深度

  3
 / \
9  20
  /  \
 15   7

输出3。

题解

复杂递归

用一个全局变量保存最大值，当可以继续递归的时候加1，然后判断是否比max大。

class Solution {
public:
    int max = 0;
    int maxDepth(TreeNode* root) {
        if(!root) return 0 ;
        dfs(root, 0);
        return max + 1;
    }
    void dfs(TreeNode* root, int now)
    {
        if(!root) return;
        if(root->left) 
        {
            dfs(root->left, now + 1);
        }
        if(root->right)
        {
            dfs(root->right, now + 1);
        }
        if(now > max)
            max = now;
    }
};

简洁递归

每次都会比较左右子树，然后累计。

class Solution {
public:
    int maxDepth(TreeNode* root) {
        //用dfs
        if(root == NULL)
            return 0;
        return 1 + max(maxDepth(root->left), maxDepth(root->right));
    }
};

617. Merge Two Binary Trees 合并二叉树

发表于 2019-03-22 更新于 2020-04-14 分类于 算法

题意

两个二叉树，将其合并并输出根节点。递归的写法很简洁，可以学习一下。

Input: 
    Tree 1                     Tree 2                  
          1                         2                             
         / \                       / \                            
        3   2                     1   3                        
       /                           \   \                      
      5                             4   7                  
Output: 
Merged tree:
         3
        / \
       4   5
      / \   \ 
     5   4   7

题解

直接相加就好了，如果不存在则返回另一个。

递归求解

class Solution {
public:
    TreeNode* mergeTrees(TreeNode* t1, TreeNode* t2) {
        //递归做，如果都存在则相加，一个不存在则返回另一个存在的。
        if(t1 == NULL)
            return t2;
        if(t2 == NULL)
            return t1;
        t1->val += t2->val;
        t1->left = mergeTrees(t1->left, t2->left);
        t1->right = mergeTrees(t1->right, t2->right);
        return t1;
    }
};

迭代解法Java版

public class Solution {
    public TreeNode mergeTrees(TreeNode t1, TreeNode t2) {
        if (t1 == null)
            return t2;
        Stack < TreeNode[] > stack = new Stack < > ();
        stack.push(new TreeNode[] {t1, t2});
        while (!stack.isEmpty()) {
            TreeNode[] t = stack.pop();
            if (t[0] == null || t[1] == null) {
                continue;
            }
            t[0].val += t[1].val;
            if (t[0].left == null) {
                t[0].left = t[1].left;
            } else {
                stack.push(new TreeNode[] {t[0].left, t[1].left});
            }
            if (t[0].right == null) {
                t[0].right = t[1].right;
            } else {
                stack.push(new TreeNode[] {t[0].right, t[1].right});
            }
        }
        return t1;
    }
}
