生命在于折腾——HHKB Type-S 简单改造分享

发表于 2020-04-13 更新于 2020-04-14 分类于 分享

0 引言

0.1 为什么买这个键盘

时间回到2018年，作者买了人生中第一台Mac，当时使用的键盘是首发买的ikbcDC-108 黑色 红轴。这个键盘说实话挺不错的，红轴手感打字也挺轻松的。但是有一天吃米酒的时候不小心倒了一点进去，导致有一些按键回弹失效生，发到售后去说：“轴体进了液体，换轴和手工费需要不少，建议买一个新的键盘，不过清理之后可以正常使用，但不保证后续正常”。基于上述原因，就有了换键盘的想法。

看来看去，听说静电容键盘手感很好，寿命很长。当时还没觉得hhkb好，因为比较贵，所以就入手了国产静电容PLUM NIZ。

用了一段时间发现这个键盘在Mac下使用有一些小问题，按键时不时会出现连击，联系了NIZ相关人员也给不了解决方案，后面只能退货处理了。当时正好看到淘宝HHKB Pro2 Type-s有活动，价格还不错一狠心就入了。不过这里出现了一些小插曲，同一款键盘我同学看到的价格和我看到的不一样，我的比他要优惠200多，这就是所谓的大数据杀熟？

分享一张当时的图片

然后一直使用良好，果然体会到了神器的不一样之处，有人可能会说：“hhkb的键位这么奇怪，怎么才能适应呢？”，针对这个问题后面会给出一些个人关于这一块的使用心得与分享。

0.2 为什么要改造键盘

疫情期间，闲的无聊在hhkb官方粉丝群日常灌水的时候，有某个小伙伴发了一张hhkb改造蓝牙的图片，当时就觉得很好奇，于是走上了键盘改造的不归路。

之前一直有听说海外有大神可以将有线版的hhkb改造成蓝牙模式，不过要价昂贵并且续航挺差。从小伙伴那得知国内有位大神在海外大神的基础上进行了改造升级，续航提升了几倍，全键编程并且还是蓝牙双模的，价格也还不贵。

HHKB2019年年底也出了一款蓝牙双模的键盘，功能描述和上面基本一致，但是看了一下价格更加劝退了 。

正好自己也一直很讨厌hhkb的那条线，在看完YDKB的键盘改造功能介绍后，直接入手了，并且加装了Type-C接口的改造（最好一步到位），这样和上面那款2699功能基本一模一样，并且更稳定，编程功能更强，外观更加好看（没有后面丑丑的电池仓），使用锂电池充电，基本不用换电池。

改造理由之一还有在张大妈上看到一张图片的猫爪键帽十分的好看 ，但是那篇文章的作者说这个hhkb专属键帽是限量的，现在已经没得卖了 。

文章链接：https://post.smzdm.com/p/alpw58q8/

某宝上只有同款的mx键帽，那么如何让HHKB的用上机械键盘的mx键帽呢？听某位群友的建议说可以换mx推杆，看了一下某宝上卖的mx推杆只能打包售卖，120元100多个。我只想换一个键帽为啥要买这么多，于是在闲鱼上花一元一个的价格淘到了别人剩下的mx推杆，改造之路开始。

群友还说“都拆键盘了，不润一下怎么行呢”，所以还入了润滑脂 。还看到某个群友po的银色logo很好看，又入了hhkb的logo金属贴纸 。

顺便说一下，有钱人可以直接淘把酷冷紫星，然后把上面的推杆换到hhkb上来，手感比某宝开模的要好很多。

1 蓝牙改造

更加详细的蓝牙改造可以看Yang大自己写的教程，比我的详细，我只是单纯分享过程。

废话不多说，直接上图。

1.1 拆键盘

hhkb键盘很好拆，卸下后面的三个螺丝即可。

这个带usb的就是老的主控，将右上角的螺丝卸下即可。拔线的时候捏住口子用力一下就行了，质量很好，不用怕断。

两款主控对比，右边中间的usb还能接一个u盘，可以直接将键盘当成一个u盘。并且解决了老款的usb不能插usb的问题，现在可以在有线的情况下直接将u盘插在键盘上。

1.2 换上ydkb主控

将主控对准后，安装即可。

上面的灯改成了即插式，用来显示低电量和层键开关状态，具体可以看某宝上的介绍。

1.3 安装电池

这里的电池也是一起购买的，个人实测每天使用10小时的情况下，能用3个星期左右。

记得在上面垫上一层泡泡膜，用来隔绝另一边的电路板，用双面胶粘贴最好。

至此蓝牙改造已经完成，是不是很简单 。将键盘开机，直接在电脑上蓝牙搜索键盘即可使用。

1.4 可编程键位

用ydkb的一大好处就是有比hhkb新款更加齐全的功能改造，最多可以定义7层键位，等于说是有了几百个键位。我最多用上3层就够了，这里分享一下自己的用法。

yang大还自己开发了网址，可以看看有没有其中一个功能能击中你的内心 ，自定义网址：http://ydkb.io/

层级触发开关，我用了右边不常用的按键当做层触发开关。

用右边的cmd加上wasd控制鼠标上下左右，q和e分别是鼠标的左键和右键，上面的1234是模拟鼠标滚轮的上下左右。这下真正实现了双手不用离开键盘，直接用键盘操控一切 。当然这个在复杂的鼠标场景下还是没有用鼠标方便，但是在看pdf论文顺便做笔记的时候很有用，不用将右手来回折腾 。

其他的层级可以自行探索，但是键盘不能触发类似Karabiner那样的应用层面的按键，只能触发硬件级别的。

2 换mx推杆

既然已经将键盘拆了，那么就继续拆下去换mx推杆。（因为到货顺序，加上自己克制不住改造的心，这里先改好了蓝牙使用了几天，然后才换的推杆 ，总共拆了好几次键盘）。

卸下键盘所有螺丝即可。

注意这个橡胶碗可能会掉下来，再重新装回去的时候要注意对齐。如果掉下来了，就将键盘用两本书架起来，将它单独安装到键盘上，然后在盖电路板上去。

将你要换的轴从正面用力推下来即可，大力出奇迹

这里我换了4个键，后续如果有需要再进行更换。

3 上润滑脂

这里我听大佬建议购买了杜邦205润滑脂，将其涂在推杆脚上，可以让键盘手感更好。具体体验是刚上手会肉不少，用了一段时间后手感渐渐变好，比起原来的声音也更加小，可以说是Type-s更静音版本。

将润滑脂刷在推杆脚的位置。

大概这个量就行了，涂太多会影响回弹，表面有一层即可表面有一层即可。

我将所有键位都拆下来，上了润滑脂。这里用一跟筷子直接捅即可，一下就能将所有的拆下来。

这里注意Types-s的静音圈别弄丢了，顺便说下非静音版本的hhkb也可以自己加装静音圈。

重新安装完毕后，紫色十字的是自己换的，黑色是hhkb特有的topre键帽。

不同样式的键帽

之所以换mx推杆，是因为静电容键盘基本上只有成团的那种键帽，不能单独定制键帽。

但是某宝上可以定制机械键盘mx轴的键帽，等于说是曲线救国了吧 。于是在某宝上定制了三个键帽，自己弄好图片发给店家即可，加上某宝买的猫爪键帽，键盘基本成型。

6 键盘使用分享

日常主力机器是2018 15寸 MBP，因为编程的缘故，自己对vim键位有一定熟悉，所以自己通过Mac上的Karabiner改键软件，通过触发vi mode，将键盘上的v和hjkl搭配实现了上下左右。

原本官方使用的s作为触发键，所谓的触发键就是当你在一定时延内同时按下s和其他function key，就能触发对应的效果。

比如触发上下左右，在100ms内同时按下s和j就是下键，然后持续按住s键，按其他hjkl就是上下左右。如果用s作为触发键可能会出现打字的时候有卡顿现象，所以我通过修改配置文件将触发键改成了汉字拼音里不会用到的v键。

Mac下找到

/Users/用户名/.config/karabiner/karabiner.json

将对应的s改成v。

当然你也可以不使用Karabiner，用ydbk通过定制组合键的形式实现上下左右，这样可以实现不同平台都能按上下左右。

7 总结

yang神这套解决方案算是hhkb有线版的终极选择了，满足了一切蓝牙双模键盘的要求，并且功能更加出色。type-s去年购买的时候价格是1700，加上改的蓝牙主控花了400多，总价大概也需要2200左右。其实和一把官方蓝牙的hhkb价格差不多，但是个人认为这一款键盘比起官方的要好看很多，因为官方的后面的电池仓是真的难看，这里没有蓝牙版本的就无法对比了。

不得不说有了蓝牙之后是真的舒服，可以把键盘放在任何地方打字，延迟方面比起之前用的ikbc双模版延迟要低很多，基本上没有感受到延迟。并且在使用了将近一个月的时间没有出现过断连现象，比起官方的蓝牙版本好太多。不要问我为什么知道，每天hhkb官方的群里时不时就有人问管理有关于键盘蓝牙的问题。

关于hhkb个人使用体验是：在打字的过程中，基本上不会移动手腕，如果熟悉了能够更加高效的输入。能通过短暂的按键学习，正确使用工具提高工作效率，这不就是人与其他动物的区别吗？科技的最本质动力也就是懒惰，人们觉得走路太累，于是有了汽车，诸如此类。hhkb的设计者不想你的手太累，于是发明了这款神奇的键盘。比如control位置，这里的设计比起之前用掌托去或者小拇指去按左下角的ctrl要舒服太多。

如果你是程序员，键盘可以说是你接触的最长时间的东西了，不用好一点的怎么行呢？

最后，可能有人会觉得这样会不会太麻烦了，这么折腾有意义么？可能大男孩的快乐就在于鼓捣这些机械的零零碎碎的小东西吧，在这个过程中获得了快乐就有意义了，快乐无价 。

最后来一张闪闪发光的HHKB

希望它能陪我走过下一个十年！
