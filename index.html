0.1 为什么买这个键盘

时间回到2018年，作者买了人生中第一台Mac，当时使用的键盘是首发买的ikbcDC-108 黑色 红轴。这个键盘说实话挺不错的，红轴手感打字也挺轻松的。但是有一天吃米酒的时候不小心倒了一点进去，导致有一些按键回弹失效生，发到售后去说：“轴体进了液体，换轴和手工费需要不少，建议买一个新的键盘，不过清理之后可以正常使用，但不保证后续正常”。基于上述原因，就有了换键盘的想法。

看来看去，听说静电容键盘手感很好，寿命很长。当时还没觉得hhkb好，因为比较贵，所以就入手了国产静电容PLUM NIZ。

用了一段时间发现这个键盘在Mac下使用有一些小问题，按键时不时会出现连击，联系了NIZ相关人员也给不了解决方案，后面只能退货处理了。当时正好看到淘宝HHKB Pro2 Type-s有活动，价格还不错一狠心就入了。不过这里出现了一些小插曲，同一款键盘我同学看到的价格和我看到的不一样，我的比他要优惠200多，这就是所谓的大数据杀熟？

分享一张当时的图片

￼

然后一直使用良好，果然体会到了神器的不一样之处，有人可能会说：“hhkb的键位这么奇怪，怎么才能适应呢？”，针对这个问题后面会给出一些个人关于这一块的使用心得与分享。

0.2 为什么要改造键盘

疫情期间，闲的无聊在hhkb官方粉丝群日常灌水的时候，有某个小伙伴发了一张hhkb改造蓝牙的图片，当时就觉得很好奇，于是走上了键盘改造的不归路。

之前一直有听说海外有大神可以将有线版的hhkb改造成蓝牙模式，不过要价昂贵并且续航挺差。从小伙伴那得知国内有位大神在海外大神的基础上进行了改造升级，续航提升了几倍，全键编程并且还是蓝牙双模的，价格也还不贵。

HHKB2019年年底也出了一款蓝牙双模的键盘，功能描述和上面基本一致，但是看了一下价格更加劝退了 。

正好自己也一直很讨厌hhkb的那条线，在看完YDKB的键盘改造功能介绍后，直接入手了，并且加装了Type-C接口的改造（最好一步到位），这样和上面那款2699功能基本一模一样，并且更稳定，编程功能更强，外观更加好看（没有后面丑丑的电池仓），使用锂电池充电，基本不用换电池。

改造理由之一还有在张大妈上看到一张图片的猫爪键帽十分的好看 ，但是那篇文章的作者说这个hhkb专属键帽是限量的，现在已经没得卖了 。

￼

方法

Go没有类，但是仍然可以在结构体类型上定义方法。

也可以对任意类型定义任意方法，但是不能对来自其他包的类型或基础类型定义方法。

方法接收者出现在func关键字和方法名之间的参数中。

func (方法接收者) 函数名() int {
}

有点像是函数指针的样子。

package main

import (
    "fmt"
    "math"
)

type Vertex struct {
    X, Y float64
}

//接收者，类似于引用传参。
func (v *Vertex) Abs() float64 {
    return math.Sqrt(v.X*v.X + v.Y*v.Y)
}


func main() {
    v := &Vertex{3, 4}
    //可以直接调用该函数
    fmt.Println(v.Abs())
}

一般方法都是用指针来接受，有两个原因：

  • 避免在每个方法调用中拷贝值；
  • 方法可以修改接收者指向的值。

其实就类似于引用，直接在参数中进行操作节约开销。

接口

有点像是java啊，c++里还没搞过接口。

接口类型是一组由方法定义的集合，可以存放实现这些方法的任何值。只有函数签名，没有具体实现。

type MyInterface interface {
    function()
}

把所有共性的方法定义在一起，任何其他类型实现了这些方法就是实现了这个接口。

方法实现规范：

func (name string) imp() string{
    print("这是实现方法的写法")
}

func sum(x int,y int) int{
    print("这是正常写法")
}

要注意函数的位置，区别方法实现和函数实现。

例子

type Phone interface {
    call()
}

type NokiaPhone struct {
}

func (nokiaPhone NokiaPhone) call() {
    fmt.Println("I am Nokia, I can call you!")
}

type IPhone struct {
}

func (iPhone IPhone) call() {
    fmt.Println("I am iPhone, I can call you!")
}

func main() {
    var phone Phone

    phone = new(NokiaPhone)
    phone.call()

    phone = new(IPhone)
    phone.call()

}

在上面的例子中，我们定义了一个接口Phone，接口里面有一个方法call()。然后我们在main函数里面定义了一个Phone类型变量，并分别为之赋值为NokiaPhone和IPhone。然后调用call()方法，输出结果如下：

I am Nokia, I can call you!
I am iPhone, I can call you!

只要方法是同名的话可以就覆盖接口，有点像是函数重载的感觉。但是具体实现还是自己写的啊，只能说根据对象的不同实现不同的函数吧。

实现泛型编程

严格来说，在Go里并不支持泛型编程，但是在c++等高级语言中使用泛型编程非常的简单。这里可以用接口实现泛型编程。

比如对自己定义的结构体进行排序，一般在C++里是要在sort函数的第三个参数给出具体排序规则。

go里的Sort函数的形参是一个interface，包含了三个方法：Len(), Less(i, j int), Swap(i,j int)。使用的时候不管数组的元素类型是什么类型，只要实现了这三个方法就可以使用Sort函数，从而达到实现”泛型编程”

这里用年龄Age对结构体Person进行排序。

type Person struct {
Name string
Age  int
}

func (p Person) String() string {
    return fmt.Sprintf("%s: %d", p.Name, p.Age)
}

// ByAge implements sort.Interface for []Person based on
// the Age field.
type ByAge []Person //自定义

func (a ByAge) Len() int           { return len(a) }
func (a ByAge) Swap(i, j int)      { a[i], a[j] = a[j], a[i] }
func (a ByAge) Less(i, j int) bool { return a[i].Age < a[j].Age }

func main() {
    people := []Person{
        {"Bob", 31},
        {"John", 42},
        {"Michael", 17},
        {"Jenny", 26},
    }

    fmt.Println(people)
    sort.Sort(ByAge(people))
    fmt.Println(people)
}

严格来说也不算是泛型编程，感觉像是这个函数方法实现。

Stringers

这个有点像是格式化输出，自己实现了stringer里的string方法。

type Stringer struct {
    String() string
}

Stringer 是一个可以用字符串描述自己的类型。fmt包 （还有许多其他包）使用这个来进行输出。

type Person struct {
    Name string
    Age  int
}

func (p Person) String() string {
    return fmt.Sprintf("%v (%v years)", p.Name, p.Age)
}

func main() {
    a := Person{"Arthur Dent", 42}
    z := Person{"Zaphod Beeblebrox", 9001}
    fmt.Println(a, z)
}

Readers

io包指定了io.Reader接口，它表示从数据流结尾读取。

io.Reader接口有一个Read方法:

func (T) Read(b []byte) (n int, err error)

Read 用数据填充指定的字节 slice，并且返回填充的字节数和错误信息。 在遇到数据流结尾时，返回 io.EOF 错误。

import (
    "fmt"
    "io"
    "strings"
)

func main() {
    r := strings.NewReader("Hello, Reader!")
    //每次循环读取8个字节，上面的字符要循环3次，最后一次的err是eof，说明读到了空。
    b := make([]byte, 8)
    for {
        n, err := r.Read(b)
        fmt.Printf("n = %v err = %v b = %v\n", n, err, b)
        fmt.Printf("b[:n] = %q\n", b[:n])
        if err == io.EOF {
            break
        }
    }
}

这个有点像是C++里的read，如果失败返回的是一个io.EOF错误。不知道这里面有没有阻塞和非阻塞的区别，或者异步同步？

Web服务器

包http通过任何实现了http.Handler的值来相应http请求：

package http

type Handler interface {
    ServeHTTP(w ResponseWriter, r *Request)
}

在这个例子中，类型Hello实现了 ‘http.Handler’，访问本地4000端口会看到来自程序的问候。

import (
    "fmt"
    "log"
    "net/http"
)

type Hello struct{}

func (h Hello) ServeHTTP(
    w http.ResponseWriter,
    r *http.Request) {
    fmt.Fprint(w, "Hello!")
}

func main() {
    var h Hello
    err := http.ListenAndServe("localhost:4000", h)
    if err != nil {
        log.Fatal(err)
    }
}

http.ListenAndServe监听某个端口，启动服务，准备接受客户端的请求。有点像是Bind后然后继续Listen，然后accpet等待请求。go将这些操作都封装到这个函数里去了。

指针

这里的指针和C里的很像，无论是定义还是使用，但是要注意的是Go里没有指针运算。

Go 具有指针。 指针保存了变量的内存地址。

定义

类型 *T 是指向类型 T 的值的指针。其零值是 nil。

var p *int

指向

& 符号会生成一个指向其作用对象的指针。

i := 42
p = &i

* 符号表示指针指向的底层的值。

解引用

fmt.Pjrintln(*p) // 通过指针 p 读取 i
*p = 21         // 通过指针 p 设置 i

这也就是通常所说的“间接引用”或“非直接引用”。

反正不能进行指针运算，比如p++之类的，会提示非法操作。

结构体

定义

关键字type 结构体名 struct，要注意命名方式。

type Vertex struct {
    X int
    Y int
}

访问

结构体使用点号来访问，和C一样。

func main() {
    v := Vertex{1, 2}
    v.X = 4
    fmt.Println(v.X)
}

结构体指针

通过指针间接的访问是透明的，前面提到的定义可以像auto一样，直接用一个变量指向另一个变量的地址就是指针了。

type Vertex struct {
    X int
    Y int
}

func main() {
    v := Vertex{1, 2}
    p := &v
    p.X = 1e9
    fmt.Println(v)
}

= 是赋值， := 是声明变量并赋值。

数组

定义

类型 [n]T 是一个有n个类型为T的值的数组。

注意这个格式

var 数组名 [数量] 类型

var a [10]int

定义变量 a 是一个有十个整数的数组。

数组的长度是其类型的一部分，因此数组不能改变大小。

切片

这里也可以在定义的时候初始化。 没有指定长度的数组就是切片，类似于c++里的vector。

func main() {
    p := []int{2, 3, 5, 7, 11, 13}
    fmt.Println("p ==", p)

    for i := 0; i < len(p); i++ {
        fmt.Printf("p[%d] == %d\n", i, p[i])
    }
}

用len来求切片的长度。

范围

可以对数组进行范围取值，表达式：

s[low:high]

表示从low到high-1的数组元素，包含两端。

func main() {
    p := []int{2, 3, 5, 7, 11, 13}
    fmt.Println("p ==", p)
    fmt.Println("p[1:4] ==", p[1:4])

    // 省略下标代表从 0 开始
    fmt.Println("p[:3] ==", p[:3])

    // 省略上标代表到 len(s) 结束
    fmt.Println("p[4:] ==", p[4:])
}

容量

切片就是一个结构体，包含了三个元素：

  1. 一个指向数组中切片指定的开始位置；

  2. 长度，即切片的长度，通过内置函数len获得；

  3. 最大长度，即切片的最大容量，通过内置函数cap获得。

扩容方式

slice这种数据结构便于使用和管理数据集合，可以理解为是一种“动态数组”，slice也是围绕动态数组的概念来构建的。既然是动态数组，那么slice是如何扩容的呢？

请记住以下两条规则：

  • 如果切片的容量小于1024个元素，那么扩容的时候slice的cap就翻番，乘以2；一旦元素个数超过1024个元素，增长因子就变成1.25，即每次增加原来容量的四分之一。
  • 如果扩容之后，还没有触及原数组的容量，那么，切片中的指针指向的位置，就还是原数组，如果扩容之后，超过了原数组的容量，那么，Go就会开辟一块新的内存，把原来的值拷贝过来，这种情况丝毫不会影响到原数组。

函数

make函数

slice 由函数 make 创建。这会分配一个零长度的数组并且返回一个 slice 指向这个数组：

a := make([]int, 5)  // len(a)=5

为了指定容量，可传递第三个参数到 make：

b := make([]int, 0, 5) // len(b)=0, cap(b)=5

b = b[:cap(b)] // len(b)=5, cap(b)=5
b = b[1:]      // len(b)=4, cap(b)=4

append

类似于vector的push_back，将元素添加到切片后面。

func main() {
    var a []int
    printSlice("a", a)
    // append works on nil slices.
    a = append(a, 0)
    printSlice("a", a)
    // the slice grows as needed.
    a = append(a, 1)
    printSlice("a", a)
    // we can add more than one element at a time.
    a = append(a, 2, 3, 4)
    printSlice("a", a)
}

如果底层数组太小，不能容纳所有值时，会分配一个更大的数组，进行自动扩容。

range

for 循环的 range 格式可以对 slice 或者 map 进行迭代循环。

var pow = []int{1, 2, 4, 8, 16, 32, 64, 128}

func main() {
    for i, v := range pow {
        fmt.Printf("2**%d = %d\n", i, v)
    }
}

返回值

Go

map

map映射键到值，必须使用make来创建；值为nil的map是空的，并且不能赋值。

type Vertex struct {
    Lat, Long float64
}

var m map[string]Vertex

func main() {
    m = make(map[string]Vertex)
    m["Bell Labs"] = Vertex{
        40.68433, -74.39967,
    }
    fmt.Println(m["Bell Labs"])
}

这里对应的值是放在中括号后面，不是hashMap里的pair键值对。

修改map

在 map m 中插入或修改一个元素：

m[key] = elem

获得元素：

elem = m[key]

删除元素：

delete(m, key)

通过双赋值检测某个键存在：

elem, ok = m[key]

如果 key 在 m 中，ok 为 true 。否则， ok 为 false，并且 elem 是 map 的元素类型的零值。

同样的，当从 map 中读取某个不存在的键时，结果是 map 的元素类型的零值。

函数闭包

Go 函数可以是闭包的。闭包是一个函数值，它来自函数体的外部的变量引用。 函数可以对这个引用值进行访问和赋值；换句话说这个函数被“绑定”在这个变量上。

例如，函数 adder 返回一个闭包。每个闭包都被绑定到其各自的 sum 变量上。

等于说是一个函数返回了另一个函数，然后可以直接将变量定义为一个函数，下面的pos就是func(x int)该函数。

func adder() func(int) int {
    sum := 0
    return func(x int) int {
        sum += x
        return sum
    }
}

func main() {
    pos, neg := adder(), adder()
    for i := 0; i < 10; i++ {
        fmt.Println(
            pos(i),
            neg(-2*i),
        )
    }
}

定义

变量定义

这里的函数变量如果是同一类型可以连续定义，比如下面这种

func add(x, y int) int {
return x + y
}

个人感觉比c++方便不少哦，不用一个变量写一个int。

多值返回

函数可以返回多个值，妈耶这个比c++好多了。

func swap(x, y string) (string, string) {
    return y, x
}
func main() {
    a, b := swap("hello", "world")
    fmt.Println(a, b)
}

这里的赋值运算不一样 是这个:=，而且也不用提前定义变量，可以直接用a，b。应该是可以直接解析成string类型吧。

占位符

如果开发者只对该函数其中的某几个返回值感兴趣的话，也可以直接用下划线作为占位符来 忽略其他不关心的返回值。下面的调用表示调用者只希望接收lastName的值，这样可以避免声 明完全没用的变量：

_, _, lastName, _ := getName()

命名返回值

返回值可以被命名，像是变量那样使用。

func split(sum int) (x, y int) {

    y = sum - x
    return
}
func main() {

返回值可以直接在函数体里操作，有点奇怪。然后直接return，函数参数后面的括号就是返回值。

定义变量

这里的变量就是var， 中间是变量名 后面是类型。

变量定义可以包含初始值，每个变量对应一个。 如果初始化是使用表达式，则可以省略类型；变量从初始值中获得类型。

101. Symmetric Tree 镜像二叉树

Posted on 2019-03-22 Edited on 2020-04-14 In 算法

题意

判断一颗二叉树是否是镜像二叉树，也就是中心对称。

For example, this binary tree [1,2,2,3,4,4,3] is symmetric:

    1
   / \
  2   2
 / \ / \
3  4 4  3
But the following [1,2,2,null,3,null,3] is not:
    1
   / \
  2   2
   \   \
   3    3

题解

递归版

利用递归思想, root相等且递归t1的左节点等于t2的右节点相等，那么才返回true，如果一个为空，另一个不为空直接返回false。

class Solution {
public:
    bool isSymmetric(TreeNode* root) {
        return mirror(root,root);
    }
    bool mirror(TreeNode* t1, TreeNode* t2){
        if(t1 == NULL && t2 == NULL) return true;
        if(t1 == NULL || t2 == NULL) return false;
        return (t1->val == t2->val) && mirror(t1->left,t2->right) && mirror(t1->right, t2->left);
    }
};

迭代版

用栈做，push两次根节点，然后取出来判断。再依次push进去t1的左节点和t2的右节点，因为相近的就是要比较的，每次弹出两个来比较。

class Solution {
public:
    vector<int> nums;
    vector<int> numsRev;
    bool isSymmetric(TreeNode* root) {
        if(root == NULL)
            return true;
        //左右一个栈，同时进行递归
        //镜面原理，将一个树分为两颗子树，他们符合一个规律。左子树等于另一个树的右子树。
        stack<TreeNode*> s;
        s.push(root);
        s.push(root);
        while(!s.empty())
        {
            TreeNode* t1 = s.top();
            s.pop();
            TreeNode* t2 = s.top();
            s.pop();
            //如果都为空则继续，可能有空节点
            if(t1 == NULL && t2 == NULL) continue;
            if(t1 == NULL || t2 == NULL) return false;
            if(t1->val != t2->val) return false;
            s.push(t1->left);
            s.push(t2->right);
            s.push(t1->right);
            s.push(t2->left);
        }
        return true;
    }
};

53. Maximum Subarray 最大的连续子数组和

Posted on 2019-03-22 Edited on 2020-04-14 In 算法

题意

给一串数字，求出连续的子数组的最大和。用O(n)解决。

Input: [-2,1,-3,4,-1,2,1,-5,4],
Output: 6
Explanation: [4,-1,2,1] has the largest sum = 6.

题解

依次累加，定义一个当前sum和最大sum。

  • 如果nowSum加了这个值是大于nums[i]的，那么就加，因为是正向的；
  • 如果nowSum加了这个值比当前的值还要小，那么这个序列是成负向的，直接把标号移动到nums[i]的位置。

因为要尽量跳过负数，所以如果减到0那么就要重新累加。

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int nowSum = nums[0];
        int maxSum = nowSum;
        for(int i = 1; i < nums.size();i++)
        {
            nowSum = (nowSum + nums[i])  >= nums[i]? (nowSum + nums[i]) : nums[i];
            maxSum = max(maxSum,nowSum);
        }
        return maxSum;
    }
};

70. Climbing Stairs 爬楼梯

Posted on 2019-03-22 Edited on 2020-04-14 In C++

题意

给你一个高度为n的台阶，每次你可以爬1层台阶，或者爬2层台阶。有多少种爬楼梯的方法。

Example 1:
Input: 2
Output: 2
Explanation: There are two ways to climb to the top.
1. 1 step + 1 step
2. 2 steps

Example 2:
Input: 3
Output: 3
Explanation: There are three ways to climb to the top.
1. 1 step + 1 step + 1 step
2. 1 step + 2 steps
3. 2 steps + 1 step

题解

动态规划，初始化1为1种，2为两种。dp[i]为前面两个dp的和。

class Solution {
public:
    int climbStairs(int n) {
        //据说是最简单的dp？
        if(n == 1) return 1;
        vector<int> dp (n+1);
        dp[1] = 1;
        dp[2] = 2;
        //从前面推后面
        for(int i = 3 ; i <= n;i++)
        {
            dp[i] = dp[i-1] + dp[i-2];

        }
        return dp[n];
    }
};

121. Best Time to Buy and Sell Stock 股票最佳买卖 — 单次买卖

Posted on 2019-03-22 Edited on 2020-04-14 In 算法

题意

一个数组，代表股票当前价格，买入和买出。怎么操作能使收益最大？只有卖掉当前股票才能买。

Example 1:
Input: [7,1,5,3,6,4]
Output: 5
Explanation: Buy on day 2 (price = 1) and sell on day 5 (price = 6), profit = 6-1 = 5.
             Not 7-1 = 6, as selling price needs to be larger than buying price.

Example 2:
Input: [7,6,4,3,1]
Output: 0
Explanation: In this case, no transaction is done, i.e. max profit = 0.

题解

菜鸡版

因为只能买卖一次，求出最大的值，然后求出最小的值。当最小值出现的时候，如果最大值的下标小于最小值的下标，那么重置最大值

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        if(prices.size() <= 1) return 0;
        int maxProfit = 0;
        int minPrice = prices[0], maxPrice = 0;
        int indexMax = 0, indexMin = 0;
        for(int i = 0;i < prices.size();i++)
        {
            if(prices[i] > maxPrice)
            {
                maxPrice = prices[i];
                indexMax = i;
            }
            if(prices[i] < minPrice)
            {
                minPrice = prices[i];
                indexMin = i;
                //当最小值出现的时候，如果最大值的下标小于最小值的下标，那么重置最大值
                if(indexMin > indexMax)
                    maxPrice = 0;
            }
            if(maxPrice - minPrice > maxProfit)
                maxProfit = maxPrice - minPrice;
        }
        return maxProfit;
    }
};

大佬版

遍历一次数组，用一个变量记录遍历过数中的最小值，然后每次计算当前值和和这个最小值之间的差值作为利润，然后每次选较大的利润来更新，当遍历完成后当前利润即为所求。

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int res = 0, buy = INT_MAX;
        for (int price : prices) {
            buy = min(buy, price);
            res = max(res, price - buy);
        }
        return res;
    }
};
